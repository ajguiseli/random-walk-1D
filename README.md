# Random Walk 1D

Este repositório contém um Jupyter Notebook que simula e plota uma caminhada aleatória (random walk) em uma dimensão usando Python e a biblioteca NumPy.

##  O que está sendo mostrado

- Implementação da random walk
- Gráfico da trajetória da partícula

##  Resultados
Resultados gráficos de uma trajetória de caminhada aleatória em uma dimensão.
